We have a function printIncrement that prints the incrementation of an Int if it is positive:

fun printIncrement(value: Int?) {
    val x: Int? = value
    if (x.isPositive()) {
        println(x + 1)
    } else {
        println("Can't increment a negative value")
    }
}
