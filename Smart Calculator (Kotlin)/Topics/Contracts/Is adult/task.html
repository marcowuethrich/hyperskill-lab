We have a function isAdult that is trying to check if a person is an adult: 

fun isAdult(value: Any?): Boolean {
    return if (value.isPerson()) {
        value.age >= 18
    } else {
        false
    }
}

fun Any?.isPerson(): Boolean {
    return this is Person
}

class Person(val name: String, val age: Int)

However, it won't compile because the compiler doesn't know that the isPerson function guarantees that value is a Person when it returns true.

Your task is to add a contract to the isPerson function that tells the compiler that value can be smartcast to Person when isPerson returns true.


Sample Input:
Imane
40

Sample Output:
Imane is an adult.


Sample Input:
Siraj
10

Sample Output:
Siraj is not an adult.
