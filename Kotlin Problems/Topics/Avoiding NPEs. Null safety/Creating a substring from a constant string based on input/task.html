Read a number as input. This number represents a string length. Then, attempt to print a substring from a constant string 'Learning Kotlin', which starts from the beginning and is as long as the inputted number suggests. If the number is bigger than the length of string, print 'Out of range'.


Sample Input:
5

Sample Output:
Learn


Sample Input:
20

Sample Output:
Out of range



Memory limit: 256 MB
Time limit: 8 seconds
