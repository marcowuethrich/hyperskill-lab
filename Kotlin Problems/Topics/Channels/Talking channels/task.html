Implement a program with two coroutines: sender and receiver. Use a channel to enable communication between the sender and the receiver. Choose an unlimited channel to ensure the sender never has to wait for the receiver.

The sender coroutine should read two messages from the user (console input) and send them to the receiver via the channel. The receiver coroutine should retrieve these messages from the channel and print "Message received: [ given input ]".


Sample Input:
Hello
User

Sample Output:
Message received: Hello
Message received: User



Memory limit: 256 MB
Time limit: 8 seconds
